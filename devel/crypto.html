Cryptocurrency¶

Cryptocurrency Quote¶

To retrieve quotes for all available cryptocurrencies, create a Stock object using a cryptocurrency ticker.

The following tickers are supported:

  • Bitcoin USD (BTCUSDT)

  • EOS USD (EOSUSDT)

  • Ethereum USD (ETHUSDT)

  • Binance Coin USD (BNBUSDT)

  • Ontology USD (ONTUSDT)

  • Bitcoin Cash USD (BCCUSDT)

  • Cardano USD (ADAUSDT)

  • Ripple USD (XRPUSDT)

  • TrueUSD (TUSDUSDT)

  • TRON USD (TRXUSDT)

  • Litecoin USD (LTCUSDT)

  • Ethereum Classic USD (ETCUSDT)

  • MIOTA USD (IOTAUSDT)

  • ICON USD (ICXUSDT)

  • NEO USD (NEOUSDT)

  • VeChain USD (VENUSDT)

  • Stellar Lumens USD (XLMUSDT)

  • Qtum USD (QTUMUSDT)

Examples¶

from iexfinance.stocks import Stock

btc = Stock("BTCUSDT")
btc.get_quote()
